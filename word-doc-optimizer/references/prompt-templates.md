# Word Document Prompt Templates

Use these templates when the user asks for a specific optimization style. Replace bracketed placeholders only when the user provides them or they can be inferred safely.

## 1. General Professional Polish

```text
请作为资深文档编辑，优化以下 Word 文档内容。

要求：
1. 不改变原意，不修改事实、数字、时间、专有名词和结论
2. 提升语言准确性、简洁度和专业感
3. 优化段落衔接与逻辑顺序
4. 删除重复表达、口语化表达和空泛措辞
5. 保留原标题、列表、步骤和层级结构

输出格式：
1. 优化后的正文
2. 本次优化说明（3-5条）
```

## 2. Formal Notice Or Announcement

```text
请把下面这份 Word 文档优化为正式通知风格，适用于[公司内部/客户通知/合作方沟通]场景。

要求：
1. 语气正式、稳妥、清晰
2. 开头直接说明事项、对象和目的
3. 正文按“背景-安排-要求-时间”展开
4. 避免口语化、模糊表述和重复语句
5. 保留所有具体日期、责任人、动作和截止时间

输出格式：
1. 可直接发出的通知正文
2. 如有必要，补充一版更简短的通知摘要
```

## 3. Report Or Proposal Optimization

```text
请优化这份报告/方案类 Word 文档，使其更适合给[管理层/客户/评审会]阅读。

要求：
1. 保留原有核心观点、数据和结论
2. 强化“背景、问题、方案、收益、风险、下一步”逻辑
3. 将长段落拆分为清晰段落或列表
4. 用更专业、更有说服力但不过度夸张的表达
5. 删除冗余背景和重复论述

输出格式：
1. 优化后的完整正文
2. 关键优化点说明
3. 如果合适，额外给出一版 200 字以内摘要
```

## 4. SOP Or Process Document

```text
请将这份流程/SOP Word 文档优化得更清晰、可执行、可培训。

要求：
1. 保留所有步骤、条件、输入输出和责任边界
2. 统一步骤表达，尽量使用动词开头
3. 将模糊描述改为明确动作或判断条件
4. 对并列步骤保持语法一致
5. 不新增原文未明确的制度或规则

输出格式：
1. 优化后的 SOP 正文
2. 对关键改动做简要说明
```

## 5. Meeting Minutes Or Recap

```text
请优化这份会议纪要 Word 文档，使其更利于会后同步和执行。

要求：
1. 保留会议结论、待办事项、责任人和时间点
2. 将分散表述整理为“议题、讨论结论、行动项”结构
3. 删除无效口语和重复叙述
4. 行动项尽量写成“负责人 + 动作 + 截止时间”形式
5. 不编造未在原文出现的决定

输出格式：
1. 优化后的会议纪要
2. 单独列出行动项清单
```

## 6. Resume Or Profile

```text
请优化这份简历/个人介绍 Word 文档，在不虚构经历的前提下，提高表达质量和职业感。

要求：
1. 不新增不存在的经历、成果或数据
2. 强化职责、成果、影响和关键词表达
3. 删除重复、空泛、自我评价式表述
4. 用更凝练的句式突出核心能力
5. 保持时间线、职位、项目名称准确

输出格式：
1. 优化后的正文
2. 如有必要，补充建议精简版
```

## 7. Aggressive Concision

```text
请将以下 Word 文档内容压缩到原长度的约 [50%-70%]，同时保留全部关键信息和结论。

要求：
1. 优先删除重复、铺垫、空话和不影响判断的信息
2. 保留事实、数据、责任、时间、结论和限制条件
3. 句子尽量短，表达尽量直接
4. 结构上保持原有阅读顺序

输出格式：
1. 压缩后的正文
2. 列出压缩策略和保留重点
```

## 8. Bilingual Business Polish

```text
请优化以下[中文/英文/中英混合] Word 文档内容，使其适合正式商务场景。

要求：
1. 保留专有名词、产品名、组织名和术语一致性
2. 中文更自然正式，英文更专业简洁
3. 避免直译腔、口语化和跨语言术语漂移
4. 原文中不明确的内容不要擅自补全

输出格式：
1. 优化后的正文
2. 如有跨语言表达需要注意的点，附 2-4 条说明
```

## 9. Paragraph Formatting Optimization

```text
请帮我优化这份 Word 文档的段落格式，不改动原意、事实、数据、日期、专有名词和结论，只做段落层面的规范化与可读性提升。

优化要求：
1. 统一段落格式，使全文风格一致、整洁、专业
2. 正文段落首行缩进 2 个字符
3. 合理调整段前、段后间距，避免忽大忽小
4. 控制行距，使阅读舒适，不拥挤也不松散
5. 段落之间层次清晰，标题、正文、列表的格式区分明确
6. 避免出现单段过长，必要时可在不改变原意的前提下合理拆段
7. 删除多余空行、重复换行、格式不一致的缩进或空格
8. 保持编号、项目符号、表格前后段落风格协调
9. 标题不设置首行缩进，正文设置首行缩进
10. 如果发现段落过密、阅读吃力，请优先通过调整分段、行距、段前段后来优化

输出要求：
1. 先给出优化建议
2. 再给出建议采用的 Word 段落参数，包括：
   - 对齐方式
   - 首行缩进
   - 行距
   - 段前
   - 段后
   - 是否分页控制/孤行控制
3. 如果原文存在明显段落结构问题，请额外指出并给出优化后的示例
4. 只做段落格式与结构优化，不额外扩写内容，不编造信息

补充说明：
这是一份正式文档，请以规范、清晰、易读为目标进行优化。
```
